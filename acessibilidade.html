Recursos Adicionais de Acessibilidade

Navegação por Teclado

Todo o site pode ser navegado usando apenas o teclado. Use Tab para avançar, Shift+Tab para voltar, e Enter para selecionar.

Leitores de Tela

Compatível com NVDA, JAWS e outros leitores de tela. Todas as imagens possuem descrições alternativas adequadas.

Estrutura Semântica

Utilizamos HTML semântico com landmarks ARIA para facilitar a navegação e compreensão do conteúdo.

Conteúdo Baixável

Oferecemos versões em PDF e texto simples de nossos conteúdos para leitura offline e uso com tecnologias assistivas.

Suporte e Feedback

Encontrou alguma barreira de acessibilidade? Entre em contato conosco para que possamos melhorar continuamente.

Design Responsivo

O site se adapta a diferentes tamanhos de tela e dispositivos, garantindo uma experiência consistente para todos.

Atalhos de Teclado

Use estes atalhos para navegar mais rapidamente pelo site

Tab Avançar para o próximo elemento
Shift + Tab Voltar para o elemento anterior
Enter Ativar link ou botão
Espaço Rolar página ou ativar controle
Esc Fechar menu ou diálogo
Home Ir para o início da página
End Ir para o final da página
↑ ↓ Navegar em listas e menus

Conformidade WCAG 2.1

Este site segue as diretrizes de acessibilidade de conteúdo web (WCAG) nível AA
